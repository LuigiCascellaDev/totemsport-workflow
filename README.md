# TotemSport

TotemSport è un plugin WordPress avanzato per la digitalizzazione
dei flussi operativi di centri di medicina dello sport.

## Features
- Workflow clinico multi-ruolo
- Integrazione POS e contante
- Generazione automatica documenti (PDF, F24)
- Cloud sync Google Drive
- Setup Wizard automatizzato

## Tech Stack
PHP, WordPress, WooCommerce, REST API, JS, Composer

## Setup (Demo)
composer install
